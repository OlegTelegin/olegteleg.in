This code from a recent data task for an RA position merges a .txt file with several .csv files into a unified dataset. It applies Regression Discontinuity Design to investigate the influence of Yelp ratings on revenue, scrutinizes data oddities, and examines potential review manipulations near rounding points. Then, it demonstrates how assumptions in data preprocessing can affect Regression Discontinuity Design outcomes.
